# PINN Beam Stiffness Reconstruction

This repository contains the implementation used in the bachelor thesis:

**"Reconstruction of the bending stiffness distribution EI(x) of Euler-Bernoulli beams using Physics-Informed Neural Networks"**

## Contents

- Python implementation of the PINN model
- Training configuration and hyperparameters
- Example plots of reconstructed EI(x) and beam deflections

## Method

The model combines:

- Physics-Informed Neural Networks
- Strong and weak form of the Euler-Bernoulli beam equation
- Control point parametrization of EI(x)
- Multi-load training
